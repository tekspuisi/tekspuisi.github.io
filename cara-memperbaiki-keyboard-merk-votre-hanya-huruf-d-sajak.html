5. 1. Terdapat sebuah keyboard yang sering digunakan. Namun tombol keyboard tersebut tidak berfungsi dengan baik. hal apakah yang harus dilakukan agar tombol keyboard tersebut dapat berfungsi dengan baik dan jelaskan penyebab permasalahan tersebut!​


Gejala Tombol Keyboard Tidak Berfungsi

Pada dasarnya kondisi tombol keybiard yang tidak berfungsi teesebut merupakan hal yang wajar, karena keyboard sering digunakan. Namun demikian, biasanya tombol keyboard yang tidak berfungsi memang sering mengganggu dan juga menyebalkan, karena menghambat pekerjaan user

Penyebab Tombol Keyboard Tidak Berfungsi

Ada beberapa hal yang menyebabkan tombol keyboard komputer tidak berfungsi. Apa saja? Berikut ini adalah beberapa hal yang dapat menyebabkan tombol keyboard komputer tidak berfungsi:

1. Kerusakan pada Beberapa Bagian Tombol yang Sering Digunakan

Penyebab pertama adalah karena beberapa tombol pada keyboard memang mengalami kerusakan. Kerusakan ini biasanya terjadi karena beberapa tombil tersebut terlalu sering digunakan. Penggunaan keyboard komputer terutama untuk keperluan tertentu seperti gaming memang memaksa beberapa tombol keyboard harus rela terus menerus digunakan. Kondisi ini menyebabkan bagian switch keyboard ataupun bagian key dari keyboard mengalami kerusakan karena terlalu sering digunakan.

Saat kerusakan tersebut di dapat karena seringnya bermain game dengan menggunakan keyboard, tentunya akan memperpendek usia hardware berjenis input ini. Maka dari itu, user perlu mewaspadai bahaya main game online di laptop dengan menggunakan keyboard ini agar kerusakan yang terjadi tidak terlalu parah sebaiknya mengurangi bermain game dengan keyboard.

Bagaimana Mengatasinya?

Untuk memperbaiki keyboard yang rusak beberapa tombolnya akibat terlalu sering digunakan, maka anda bisa melakukan pengistirahatan tombol tersebut. Paling tidak, hindari penggunaan tombol tersebut untuk sementara waktu, agar tombol tersebut bisa “beristirahat”.

Apabila hal ini tidak berhasil, maka apa boleh buat, anda harus rela mengganti keyboard, karena kemungkinan kerusakan sudah menjadi sangat parah, dan sulit diatasi.

2. Kerusakan pada Kabel dan Socket Keyboard

Baik komputer desktop atau PC, serta laptop dan juga notebook, sama sama menggunakan kabel dan juga socket untuk menjalankan fungsi inputnya. Namun terkadang, seringkali terjadi kerusakan pada socket dan juga sistem perkabelan dari keyboard tersebut.

Ciri – ciri Kerusakan pada Babel dan Socket Keyboard

Biasanya, kerusakan pada socket dan juga kabel pada keyboard menimbulkan gejala dimana keseluruhan keyboard tidak dapat difungsikan sama sekali, meskipun keyboard sudah tersambung dengan benar.

Bagaimana Mengatasi Hal Ini?

Untuk mengatasi hal ini, anda bisa membawa keyboard anda kepada ahli maintance hardware komputer, untuk dilakukan penggantian pada bagian kabel dan juga socket keyboard anda.

3. Keyboard Terjatuh, Terbanting atau Terbentur Benda Keras

Penyebab lainnya dari tombol keyboard yang tidak dapat befungsi adalah karena keyboard komputer anda terjatuh, terbanting, ataupun terbentur benda keraa hingga parah. Hal ini biasanya cukup sulit untuk diperbaiki, karena kemungkinan keseluruha sistem komponen elektronik di dalam keyboard anda mengalami kerusakan total, sehinnga lebih direkomendasikan bagi anda untuk mengganti keyboard komputer tersebut

4. Kemasukkan Debu dan Benda Asing Lainnya

Ini merupakan penyebab tombol keyboard tidak berfungsi yang paling sering terjadi. Ya, terkadang kebersihan dari keyboard jarang sekali diperhatikan, sehingga kita sebagai user tidak menyadari, bahwa sebenarnya keyboard komputer kita sangat kotor oleh debu dan benda asing lain, seperti kerikil kecil dan bahkan remah – remah makanan.

Cara mengatasinya:

Untuk mengatasi hal ini ada dua cara yang bisa anda lakukan dengan mudah. Berikut ini adalah cara untuk mengatasi permasalahan tombol keyboard tidak berfungsi akibat debu dan benda asing lainnya :

Bersihkan keyboard anda dengan kuas dan juga mini vacuum cleaner. Hal ini sangat pas untuk dilakukan bagi anda yang tidak berani membongkar keyboard anda.

Cara kedua, anda bisa membongkar keyboard anda dan membersihkannya. Cara ini adalah cara yang paling efektif, karena benar benar dapat membersihkan keyboard laptop dan komputer anda secara tuntas.

30. dalam tata papan keyboard terdapat beberapa tombol huruf dan lampu indikator yang menandakan keyboard berfungsi dengan baik dan jelaskan susunan tombol keyboard tersebut​


Jawaban:1. Main Typing keypad

Main typing keyopad adalah papan ketik utama yang berisi huruf A sampai Z, tanda baca dan angka 1 – 0 yang digunakan untuk memasukkan teks ke dalam sistem komputer misalnya saat mengetik pada Microsoft Office atau pada browser internet.

2. Tombol Penunjuk Arah

Dalam keyboard posisi direction keys ini biasanya di sudut kanan bawah. Tombol itu biasanya berisi petunjuk arah dengan simbol anak panah.

3. Numeric/ Directional Keypad

Numerik atau NUM, adalah tombol yang berisi angka dan tombol pemindah kursor yang biasanya terletak pada sisi kanan main typing board.

Tombol ini hanya akan berfungsi jika tombol fungsi Numeric key dinyalakan. Tombol ini akan memudahkan seseorang dalam mengetik dan memasukkan angka ke dalam sistem.

4. Function Keys

Tombol-tombol ini berada diatas tombol main typing keypad dan dapat dikenali dari tanda F1 hingga F12. Tombol-tombol ini berfungsi untuk mengatur beberapa perintah seperti menyimpan dokumen, memunculkan menu help saat mengetik di Microsoft word dan lain sebagainya.

5. Special Keys

Tombol ini ada dibagian atas tombol directional keys dan terdiri dari tombol Home, Delete, Page up, Page Down, Insert, Print screen, pause break dan End.

6. Modifier Keys

Mofofier keysYang termasuk dalam golongan keypad ini adalah tombol Ctrl, Alt dan Shift. Sesuai dengan namanya, ketiga tombol ini akan merubah atau memodifikasi fungsi suatu tombol lain dalam keypad dan bisa ditekan bersamaan tombol lain tersebut untuk memunculkan suatu perintah.

Misalnya tombol Ctrl yang ditekan bersamaan dengan huruf C akan secara otomatis menyalin data yang sudah dipilih pada monitor. Kombinasi tombol-tombol tersebut juga sering disebut sebagai shortcut keys.

8. Lock Keys

Lock keyas adalah indikator cahaya yang menandakan bahwa suatu tombol sedang digunakan untuk mengatur suatu fungsi misalnya tombol Caps lock yang memungkinkan data diketik dengan huruf kapital dan numeric lock yang digunakan untuk mengaktifkan tombol numeric key.

9. Windows keys

Biasanya pada keyboard pada komputer windows memiliki tombol yang bergambar logo Microsoft windows itu sendiri. Tombol ini berfungsi untuk memunculkan menu pada jendela windows pada komputer.

35. Apa artinya Votre chien


Artinya: Anjlng-mu.

Votre = kata kepemilikan (-mu).

Le chien = anjlng.

Pembahasan

Kata kepemilikan merupakan suatu kata yang menyatakan bahwa suatu barang atau benda milik seseorang, misalnya -milikku, -milikmu, -miliknya, dan lain-lain.

Votre merupakan kata kepemilikan untuk subjek vous yang artinya Anda atau kalian (*formal) dan umum dipakai bisa untuk benda masculin dan feminin.

Chien artinya Anjlng, karena chien adalah benda masculin tunggal maka kata sandang yang tepat adalah un atau le.

Seorang editor koran asal Inggris bernama Chrisptopher Scholes adalah penemu tata letak unik itu tahun 1860-an. Bukan temuan sepele, karena dia meraup banyak uang dari hak paten. Model ini diberi nama QWERTY, sesuai urutan huruf pada baris paling atas. Model itu digunakan secara luas di seluruh dunia termasuk Indonesia.

Pada awal penemuan, tahun 1714 tata letak mesin ketik dibuat urut (berdasar huruf latin). Apakah menurutmu tata letak keyboard mestinya demikian? Dibuat urut untuk memudahkan mencari huruf ketika mengetik?
Sepintas tata letak yang urut memudahkan. Terutama bagi orang yang mengetik menggunakan ”sebelas jari”. Namun para juru ketik profesional, yang mengetik menggunakan sepuluh jari justru kesulitan.

Mengetik dengan sepuluh jari lebih cepat. Bisa mencapai 60 kata per menit. Namun memang tidak mudah, karena jari mesti hafal letak huruf. Pandangan mata fokus pada naskah, sedangkan jari bergerak sendiri menyusuri tombol.

Memang tak mudah jika belum terbiasa. Tetapi jika sudah mahir, jari akan bergerak secara reflek dan otomatis. Jaman dahulu, kursus mengetik mahal biayanya. Saat belajar mata ditutup kain agar tak bisa melihat mesin ketik. Cara ini disebut ”mengetik buta”.

Huruf-huruf pada mesin ketik disusun sedemikian rupa, yang sering digunakan bersama-sama diletakkan berdekatan. Misalnya, huruf A dan S sering digunakan bersama untuk membentuk kata, maka posisinya dibuat bersebelahan. Sehingga jari bisa lebih cepat menjangkau.

Jika dibuat urut abjad justru menyusahkan. Huruf yang sering digunakan secara bersama-sama, letaknya malah berjauhan. Demikian pula sebaliknya. Misalnya huruf A, B, dan C letaknya berurutan. Padahal, ketiganya jarang digunakan bersama-sama dalam satu kata. Kamu jarang menemui kata yang mengandung sekaligus tiga huruf A, B, dan C kan?

Untuk Orang Kidal
Huruf yang sering digunakan diletakan pada baris tengah. Baris ini digunakan tumpuan jari saat mengetik. Sebenarnya, tata letak ini didesain untuk orang kidal. Huruf-huruf yang sering digunakan diletakkan pada kolom kiri. Maklum, orang Eropa banyak yang kidal. Tangan kirinya lebih aktif, termasuk saat mengetik.

Lay-out QWERTY lebih sesuai untuk kosakata Inggris. Beberapa negara seperti Belgia, Perancis, Jerman, dan Norwegia menciptakan tata letak sendiri yang lebih sesuai dengan kosakata bahasa mereka.
Lay-out QWERTY bukanlah yang terbaik. Seorang ekonom bernama Dr August Dvorak menciptakan tata letak baru dan dipatenkan tahun 1936. Model ini lebih efisien.
Jaman dahulu sering diadakan lomba mengetik. Pesertanya para sekretaris dan juru ketik profesional.

Pemenang biasanya menggunakan mesin ketik Dvorak. Guiness Books of World Record mencatat rekor mengetik tercepat dipegang Barbara Blackburn pada 2005. Sungguh mengagumkan, dia mengetik dengan kecepatan 170 kata permenit menggunakan mesin ketik Dvorak.
Meski lebih efisien, model Dvorak tak populer. Orang terlanjur familier model QWERTY ciptaan Christopher Scholes. Kebanyakan keyboard komputer juga menggunakan model ini, kecuali Apple Macintosh yang menggunakan Dvorak.
