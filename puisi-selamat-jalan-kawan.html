16. Selamat malam kawan. Mohon bantuan kalian


matriks sb y . matriks transformasi . (x,y) = (x',y')
(-1 ... 0) (2 ... 0) (x) = (x')
(0 ... 1). (-1 ... 3) (y) ... (y')

(-2 ... 0) (x) = (x')
(-1 .... 3) (y) ... (y')

(-2x + 0y) = (x')
( -x + 3y ) ....(y')

-2x = x' => x = x'/-2

-x + 3y = y'
=> 3y = y' + x
=> 3y = y' + x'/-2
=> y = (1/3)y' - (1/6)x'

jadi bayangan dari 4x - y + 5 = 0 adalah
4(x'/-2) - (1/3 y' - 1/6 x') + 5 = 0 ..... kali 6
=> -12x' - 2y' + x' + 30 = 0
=> -11x' - 2y' + 30 = 0
=> 11x + 2y - 30 = 0
