8. Kegiatan berkemah anak pramuka diikuti 9 orang yabg duduk mengelilingi api unggun jika 3 orang selalu duduk berdekatan berapa banyak cara mereka dapat duduk mengelilingi api unggun tersebut?


Pada kegiatan berkemah, 9 anak pramuka sedang duduk mengelilingi api unggun. Jika 3 orang selalu duduk berdekatan, maka banyak cara mereka dapat duduk mengelilingi api unggun tersebut adalah 4.320 cara.

Penjelasan dengan langkah-langkah

Untuk menyelesaikan soal di atas, kita dapat menggunakan rumus permutasi siklis yaitu banyak cara n orang duduk melingkar adalah:

(n – 1)!

Diketahui

Banyak anak yang mengelilingi api unggun = 9 orangBanyak anak yang selalu duduk berdekatan = 3 orang

Ditanyakan

Tentukan banyak cara mereka dapat duduk mengelilingi api unggun tersebut!

Jawab

Langkah 1

Dalam soal diketahui bahwa 3 orang selalu duduk berdekatan berarti:

Sisa orang yang duduknya bebas dimana saja = 9 – 3 = 6.3 orang yang selalu duduk berdekatan dianggap 1.

Jadi diperoleh:

n = 6 + 1 = 7

Langkah 2

Banyak posisi duduk 3 orang yang selalu duduk berdampingan adalah:

3!

Langkah 3

Berdasarkan langkah 1 dan 2, banyak cara 9 orang duduk melingkar dengan syarat 3 orang selalu duduk berdampingan adalah:

= (n – 1)! × 3!

= (7 – 1)! × 3!

= 6! × 3!

= (6 × 5 × 4 × 3 × 2 × 1) × (3 × 2 × 1)

= 720 × 6

= 4.320

Kesimpulan

Jadi banyak cara 9 orang duduk melingkar dengan syarat 3 orang selalu duduk berdampingan adalah 4.320 cara.

Q = 9,12x10⁵ J/s

Perpindahan secara radiasi :
Q/t          = eσ A T⁴
9,12x10⁵ = 1. 5,67.10⁻⁸ . A (360⁴ – 300⁴) 
9,12x10⁵ = 493,07 x A
A             = 9,12x10⁵/493,07
A             = 1,85x10² m²

anak-anak mengitari, jadi :
A = π x r²
r² = 1,85x10²/3,14
r² = 0,59x10²
r  = √59

Permutasi siklis adalah banyaknya cara menyusun suatu objek yang posisinya melingkar.

P = ( n - 1 ) !
