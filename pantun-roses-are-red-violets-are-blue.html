22. Lucy has a necklace which has red beads and blue beads. For every red bead, there are four blue beads. There are 20 beads in total. How many of these are blue?


Jawaban:

Let's denote the number of red beads as "R" and the number of blue beads as "B."

From the information given, we can create two equations:

   R + B = 20 (There are 20 beads in total).

   For every red bead, there are four blue beads: R = 4B.

Now, we can use these equations to solve for the number of blue beads (B).

First, substitute the value of R from the second equation into the first equation:

4B + B = 20

Combine like terms:

5B = 20

Now, divide both sides by 5 to find the number of blue beads (B):

B = 20 / 5

B = 4

So, there are 4 blue beads.
