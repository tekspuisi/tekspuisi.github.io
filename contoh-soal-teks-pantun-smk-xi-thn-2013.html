5. tolong di bantu yah, soal matematika SMK kelas XI ​


a. Persamaan lingkaran berpusat (-2,5), melalui (-1,2)

(x + 2)² + (y - 5)² = r²

→ (-1 + 2)² + (2 - 5)² = 10

→ x² + y² + 4x - 10y + 4 + 25 - 10 = 0

→ x² + y² + 4x - 10y + 19 = 0

b. Persamaan lingkaran berpusat (-2,5) menyinggung garis x - 3 = 0 → x = 3

(x + 2)² + (y - 5)² = r²

→ (3 + 2)² + (0 - 5)² = r²

→ r² = 25

→ (x + 2)² + (y - 5)² = 25

→ x² + y² + 4x - 10y + 4 + 25 - 25 = 0

→ x² + y² + 4x - 10y + 4 = 0

14. Tuliskan rumus persamaan lingkaran kelas xi smk dan contohnya, buat persentasi besok, syukron


Rumus persamaan lingkaran tergantung kepada

nilai titik pusat ( a,b) dan besarnya jari-jari

lingkaran.

jari - jari lingkaran yang merupakan selisih jarak

antara titik pusat dengan suatu titik ( x,y)

yang merupakan tepi dari lingkaran/ melalui titik

tersebut.

dimisalkan :

suatu lingkaran dengan titik pusat P (a,b) melalui

titik Q ( x,y ) dirumuskan  :

jarak PQ : ( jari-jari )

PQ² = (x - a)²+ (y-b)²

sehingga

r² =  (x - a)²+ (y-b)²


contoh soal 1):

Tentukan persamaan lingkaran yang berpusat

di (2, 3) dan berjari-jari 6!

Jawab :

P (a,b) --> ( 2,3)

r = 6

r² =  (x - a)²+ (y-b)²

6² = (x-2)² + (y-3)²

36 =  (x-2)² + (y-3)²

Jadi persamaan lingkarannya:

(x-2)² + (y-3)² = 36

contoh soal 2):

Tentukan persamaan lingkaran yang

berpusat di (1,2) dan melalui titik (4,-3)!

persamaan lingkaran berpusat di (1,2)

dengan  r adalah :

(x-1)² + (y-2)² = r²

maka untuk mencari r melalui titik (4,-3)

(x-1)² + (y-2)² = r²

(4-1)² + (-3-2)² = r²

(3)² + (-5)² = r²

9 + 25 = r²

34 = r²

jadi persamaamn lngkarannya :

(x-1)² + (y-2)² = 34

28. contoh teks anekdot siswa smk


Mencuri Sendal

Di suatu pagi, Rahmat lagi asyik menyantap soto di warung makan favoritnya. Seusai merasa kenyang, Rahmat langsung berdiri dan bergegas pulang.

Di tengah tengah berpergian untuk pulang, Rahmat tertimpa musibah berupa kecelakaan karena kena serempet sepeda motor yang lagi ugal-ugalan. Akibat kecelakaan itu, akhirnya putus sandal Rahmat.

Secara terpaksa Rahmat berjalan kaki dengan tidak disertai memakai sandal. Dikarenakan jarak rumahnya yang jauh, akhirnya dia memilih mau berkunjung ke toko terdekat buat membeli sandal. Namun, apa buat, jumlah uang di dompetnya tidak cukup.

Karena jumlah uang tak cukup, Rahmat akhirnya memiliki niatan buat mengambil sandal di masjid yang lokasinya cuma beberapa meter dari toko sandal tersebut. Rahmat akan mengambil sandal paling bagus yang terdapat di masjid tersebut.

Dengan duduk di teras masjid, ia sambil juga melihat tiap orang yang akan memasuki ke masjid. Sehingga nantinya saat incarannya sedang sibuk ketika beribadah ia segera mengambil sandal tersebut.
