11. Rizal menabung di bank sebesar Rp2.100.000,00 dengan suku bunga tunggal sebesar 8% pertahun. Saat diambil, tabungan Rizal menjadi Rp2.282.000,00. Lama Rizal menabung adalah ....​


tabungan awal = Rp2.100.000

tabungan akhir = Rp2.282.000

p = 8%

tabungan akhir = tabungan awal + n/12 x p x tabungan awal

2.282.000 = 2.100.000 + n/12 x 8/100 x 2.100.000

2.282.000 - 2.100.000 = n/12 x 8 x 21.000

182.000 = n/3 x 2 x 21.000

182.000 = n/3 x 42.000

182.000 x 3 / 42.000 = n

13 = n

Jadi, lama Rizal menabung adalah 13 bulan

37. Tanah ocu iman berbentuk persegi panjang luasnya 1296 meter2. Jika lebarnya 27 meter. Berapa meter keliling tanah ocu iman....A. 75 meter B. 130 meter C. 150 meterD. 170 meter​


Jawab:

C

Penjelasan dengan langkah-langkah:

Luas (L) = p x l

p x 27 = 1296

p = 1296/27

p = 48 meter

Keliling = 2 (p + l) = 2 (48 + 27) = 2 x 75 = 150 meter (C)

Diket = luas = 1.296 m^2

lebar = 27 meter

Ditanya = Keliling tanah ocu iman

Penyelesaian = Pertama kita harus mencari panjang dari persegi panjang.

*Cara mencari panjangnya*

= 1.296 m ÷ 27 m

= 48 m

Setelah panjang sudah di ketahui, kita cari kelilingnya

*Cara mencari kelilingnya*

= 2 × (p + l)

= 2 × (48 m + 27 m)

= 2 × 75 m

= 150 m

Jawabannya adalah C
