Jawab:

Santi = 96

Rani = 72

Leni = 90

Penjelasan dengan langkah-langkah:

Santi : Rani = 8 : 6 ( ×2)

Rani : Leni = 4 : 5 ( ×3)

Santi : Rani : Leni = 16 : 12 : 15

Rani = 18 + Leni

Rani - Leni = 18

Rani = 12/(15 - 12) x 18 = 12/3 x 18 = 72

Santi = 16/3 x 18 = 96

Leni = 15/3 x 18 = 90

4. kulit kuaci mengandung apa ?


Pengertian kuaci :

Kuaci adalah camilan berupa biji semangka, biji waluh, atau biji bunga matahari yang dikeringkan dan diasinkan. Proses penyangraian memberikan aroma sedap, dan penambahan garam memberikan rasa gurih dan meningkatkan daya awet.

kulit kuaci :

Kulit kuaci memiliki banyak kesamaan dengan kandungan kulit kacang yaitu N 2,64%, P2O5 3,56%, K2O 1,67% dan C-organik 4,93 dengan pH 6,95.

Manfaat kuaci :

Paparan Sinar matahari yang langsung ke kulit wajah dapat menyebabkan wajah terlihat kusam. Namun dengan mengonsumsi rutin, kuaci tersebut bisa diatasi dan membuat kulit wajah lebih bercahaya dan glowing. Selain itu kandungan vitamin E pada kuaci juga mampu membuat kulit tetap sehat dan lebih kencang.

19. jumlah kuaci santi berbanding kuaci rani adakah 8:6 banyak kuaci rani sama dengan 4/5 kuaci leni.kuaci rani lebih banyak 18 buah dari kuaci leni.berapa jumlah kuaci danti,rani dan leni seluruhnya?


Misalkan :
santi = x
rani = y 
leni = z
Penyelesaian :
Dari data di soal kita membuat persamaan :
Perbandingan kuaci santi dan rani :
x : y  = 8 : 6

Kuaci rani sama dengan 4/5 kuaci leni :
y = 4/5 z

kuaci rani lebih banyak 18 buah dari kuaci leni :
y+18 = z

Sehingga dengan persamaan diatas :
y = 4/5 z 
z = (y)/(4/5)
z = 5/4y

Subtitusi z = 5/4y ke persamaan :
y+18 = z
y+18 = 5/4y
y - 5/4y = -18
- 1/4y = - 18
1/4y = 18
y = 72

Sehingga untuk nilai z :
72 = 4/5 z
z = 5/4 x 72 
z = 360/4
z = 90

Maka untuk nilai x :
x : y = 8 : 6
x : 72 = 8 : 6
x = 8/6 x 72 
x = 576
x = 96 

Sehingga Untuk Jumlah kuaci seluruhnya :
JK = santi + rani + leni  
JK = x + y + z
JK = 96 + 72 + 90 
JK = 258 biji

Jadi jumlah kuaci seluruhnya santi rani dan leni ialah 258 bijis : r = 8 : 6

Jawaban:

Pertama biji-biji labu anda rendam ke dalam air kapur kemudian dikeringkan sampai benar-benar kering . Setelah itu garam, kayu manis,serai, pandan wangi bersama-sama anda masukkan ke dalam air dan kemudian dididihkan. Lalu kurang lebih setengah jam, masukkkan biji biji tersebut dan diaduk, sampai biji-biji tersebut cukup mudah untuk di pecahkan . Setelah itu tiriskan dan keringkan.

Jika kamu punya 100 gram biji labu, campurlah biji labu ini dengan 25 gram garam. Masukkan keduanya dalam panci dan tuang air kira-kira dua gelas besar. Aduk biji labu dan garam hingga garam larut. Jika sudah, rebus biji kuaci tersebut selama beberapa menit dengan api sedang dan air di dalam panci mendidih serta berkurang. Jika air sudah berkurang banyak, angkat panci dan tiriskan biji labu. Setelah itu, keringkan biji labu di bawah sinar matahari langsung hingga kering dan keras. Biji kuaci pun sudah jadi dan siap untuk dijadikan camilan.

Jika kamu membuat biji kuaci ini saat musim penghujan dan sulit mendapatkan sinar matahari langsung. Kamu bisa menggoreng biji labu dengan cara goreng sangrai (goreng tanpa minyak). Gimana, untuk membuat camilan biji kuaci sendiri di rumah sangat mudah bukan? Jika kamu mengolah labu, jangan langsung buang isinya karena isinya bisa dijadikan biji kuaci atau camilan enak. Selamat mencoba dan semoga tips ini bermanfaat.

37. jumlah kuaci santi berbanding kuaci rani adalh 8:6 dan banyak kuaci rani sama dengan 4/5 kuaci leni.kuaci rani lebih banyak 18 buah dari kuaci leni.berapa jumlah kuaci santi,rani dan leni seluruhnya?tolong dijawab pake caranya


santi = a
rani = b 
leni = c

a:b = 8:6
b = 4/5 c
b+18 = c

karena b = 4/5c,c = 5/4b
b+18 = 5/4b
1/4b = 18
b=72

72 = 4/5 c
c = 5/4 x 72 = 90

a : b = 8 : 6
a : 72 = 8 : 6
a = 8/6 x 92 = 96 

santi + rani + leni = 96 + 72 + 90 = 258


38. Pak Wiliam adalah seorang pengusaha kuaci, setiap satu kemasan kuaci produksi pak wiliam memiliki berat bersih 50 gr dengan berat pembungkusnya 2 gr. Berat kotor kuaci tersebut dalam satu kemasan adalah​


Jawaban:

52 gram

Penjelasan dengan langkah-langkah:

Diketahui :

Neto = 50 gr

Tara = 2 gr

Ditanya : Bruto?

Dijawab :

Bruto = Neto + Tara

Bruto = 50 + 2

Bruto = 52 gr

Jadi, berat kotor kuaci dalam satu kemasan adalah 52 gram.
