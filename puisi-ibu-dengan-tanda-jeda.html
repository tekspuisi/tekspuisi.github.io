Catatan :

Penjedaan puisi adalah waktu berhenti sejenak, agar makna dari puisi dapat tersampaikan dari pembaca agar dapt diresapi oleh pendengar, selain itu untuk memperindah bahkan mempermudah cara baca puisi.

/ -> sebentar seperti (,) koma

// -> berhenti seperti (.) titik

/// -> berhenti lama bahkan bisa garis miring empat ////

11. Jeda pada puisi biasanya menggunakan tanda .....


Tanda jeda pada puisi biasanya menggunakan tanda seperti :

1. Tanda jeda / artinya berhentu sebentar untuk bernapas, biasanya pada koma atau di tengah baris. 

2. Tanda jeda // artinya berhenti agak lama, biasanya koma di akhir barus yag masih berhubungan dengan baris berikutnya.

Jeda pada puisi biasanya menggunakan tanda Koma (,) Tanda jeda // artinya berhenti agak lama
