Total Persentase : Olahraga + Melukis + Puisi + Menulis

100% = 40% + 15% + 10% + x (misal, anak gemar Menulis = x)

100% = 65% + x, maka...

x = 35%

Diketahui, olahraga = 40% = 32 anak

Apabila total anak-anak = a, maka...

40% x a = 32 anak

40/100 x a= 32 anak

a = 80 anak

Sehingga, anak-anak yang gemar menulis adalah...

80 x 35%

= 80 x 35/100

= 28 anak

360° = 15° + 45° + 66° + 60° + x (misal, PNS = x)

360° = 186° + x

x = 174°

Petani,

= 240 x 15/360

= 10 orang

TNI,

= 240 x 45/360

= 30 orang

Polri,

= 240 x 66/360

= 44 orang

Pedagang,

= 240 x 60/360

= 40 orang

PNS,

= 240 x 174/360

= 116 orang
