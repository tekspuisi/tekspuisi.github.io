4. conto paguneman sunda


Paguneman nyaeta ngawangkong atawa ngobrol dua arah, antar dua urang atawa leuwih, silih tempas, jeung kalimah anu digunakeun dina paguneman ngagunakeun kalimah langsung. Umpama ditilik tina suasanana, aya dua rupa paguneman, nyaeta paguneman resmi jeung paguneman teu resmi. Conto paguneman resmi nyaeta saperti sawala atawa diskusi. Sedengkeun conto paguneman henteu resmi nyaeta saperti ngawangkong jeung babaturan atawa kolot sorangan.

Dina paguneman aya sababaraha aturan anu kudu diperhatikeun, nyaeta anu aya patalina jeung tarik launna sora atawa lafal, luhur handapna sora atawa lentong, pilihan kekecapan anu digunakeun dina paguneman, sarta penjiwaan dina ngalakukeun paguneman. Unsur pilihan kekecapan, dina basa Sunda kaasup anu cukup penting, sabab dina basa Sunda aya dua ragam basa, nyaeta Ragam Basa Hormat / Lemes jeung Ragam Basa Loma.

Umpama nilik kana jawaban anu geus aya, nyaeta paguneman antara dokter jeung pasien, eta paguneman teh bisa kaharti jeung jelas maksudna. Tapi umpama nilik kana ragam basa anu digunakeun, eta ragam basa teh kurang pas sabab ngagunakeun Ragam Basa Loma. Umumna paguneman antara pasien jeung dokter biasa ngagunakeun Ragam Basa Hormat / Lemes.

Ragam Basa Loma biasa digunakeun pikeun nyarita jeung babaturan anu geus akrab atawa loma. Di handap ieu conto paguneman basa Sunda antar barudak anu geus akrab atawa loma anu ngagunakeun Ragam Basa Loma:

6. mengapa kita harus bangga dengan adanya aksara sunda?​


Jawaban:

Kita dapat merasa bangga dengan adanya aksara Sunda karena aksara tersebut adalah bagian dari warisan budaya dan sejarah yang kaya dari suku Sunda di Indonesia. Berikut adalah beberapa alasan mengapa Anda bisa merasa bangga:

1. Identitas Budaya: Aksara Sunda merupakan salah satu bentuk identitas budaya suku Sunda.

2. Preservasi Sejarah: Aksara Sunda adalah salah satu bentuk peninggalan sejarah yang berharga.

3. Bahasa dan Sastra: Aksara Sunda membantu mempertahankan kekayaan bahasa dan sastra Sunda.

4. Menghormati Leluhur: Dengan mempelajari dan menggunakan aksara Sunda, Anda menunjukkan penghargaan kepada leluhur Sunda yang telah menciptakan dan mengembangkan sistem penulisan ini.

5. Pendidikan dan Pengembangan Pribadi: Mempelajari aksara Sunda dapat meningkatkan pengetahuan dan keterampilan pribadi Anda.

Mengingat nilai-nilai ini, memelihara dan menghargai aksara Sunda adalah hal yang patut dibanggakan, karena itu berarti Anda peduli terhadap warisan budaya dan sejarah yang kaya dari suku Sunda dan memainkan peran dalam menjaganya agar tetap hidup dan relevan dalam masyarakat modern.

15. tuliskeun conto artikel sunda


Henteu jarang urang sok bingung umpama dititah kudu nyieun hiji artikel. Biasana, urang sok bingung teh dina naon anu rek ditulis. Kumaha mimitina, jeung naon eusina. Kitu deui umpama urang geus mimiti nulis artikel jeung nulis eusina. Kadang-kadang sok bingung iraha urang kudu eureun nulis eta artikel.

Pikeun nulis hiji artikel, aya hiji padoman anu basajan jeung gampil ngagunakeunana. Malahan eta padoman teh lain ngan pikeun nulis artikel wae, tapi pikeun nulis tulisan naon wae. Padoman anu dimaksud nyaeta padoman atawa rumusan anu mindeng disebut 5W+1H. Rumusan 5W+1H teh nyaeta padoman anu nyebutkeun yen hiji artikel bisa disebut cukup lengkep umpama eta artikel miboga unsur-unsur anu disebut 5W+1H. Unsur-unsur 5W+1H nyaeta what, who, when, where, why jeung how.

Umpama dina nyieun hiji artikel, urang geus bisa nyumponan sakabeh unsur-unsur anu aya dina 5W+1H, sahenteuna geus bisa dianggap yen artikel anu ku urang dijieun eta teh lengkep. Umpama hiji artikel geus bisa dianggap lengkep, tinggal anu kudu dilakukeun teh nyaeta nyusun kecap-kecap atawa kalimahna anu merenah sangkan eta artikel teh bisa kaharti ku anu maca. Di handap ieu contoh artikel basa Sunda anu ngagunakeun atawa nerapkeun rumusan 5W+1H :

Kareta Api Jakarta-London

Bisa jadi lain ngan sakadar ngimpi atawa ngalamun, kahareup mah urang aya kamungkinan bisa naek kareta api ti Jakarta nepi ka London. Naon sababna lain ngan sakadar ngimpi?. Sabab kiwari geus aya kaciri tanda-tandana.

Dina raraga nyanghareupan Masyarakat Ekonomi ASEAN, geus aya wacana perluna diwangun hubungan kareta api cepat antar nagara di ASEAN. Salian ti eta, nagara Tiongkok ayeuna geus bisa ngajalankeun kareta api barangna nepi ka kota London anu ngabutuhkeun waktu perjalanan salila 18 poe.

Terus kumaha carana pikeun ngahubungkeun kareta api nagara-nagara ASEAN ka Tiongkok ?.

Para pamingpin kareta api sa ASEAN geus rengse ngayakeun pertemuan anu ngabahas wacana perluna diwangun hubungan kareta api cepat antar nagara-nagara nu aya di ASEAN. Eta pertemuan nu dihadiran ku para pamingpin kareta api sa ASEAN teh geus rengse diayakeun di bulan Oktober 2016. Pertemun para pamingpin kareta api sa ASEAN eta teh diayakeunana di stasiun jeung musium kareta api Indonesia nu ayana di kota Ambarawa, Semarang.

Dina eta pertemuan teh, salian ti rencana hubungan kareta api cepat di nagara-nagara ASEAN, oge disinggung kamungkinan ayana hubungan kareta api cepat ti nagara Tiongkok ka nagara-nagara ASEAN. Contona ceunah, ayeuna di Tiongkok teh geus aya kareta api cepat ti kota Beijing ka kota Guangzhou anu diharepkeun dina engkena bisa ngahubung oge ka nagara-nagara nu aya di ASEAN, kaasup kota-kota nu aya di Indonesia.

Engke teh ceunah kareta api ti Tiongkok anu ka nagara-nagara ASEAN, bisa turun di nagara-nagara Vietnam, Thailand, Malaysia terus nepi ka Indonesia, ngaliwatan ka laut, terus asup ka kota Dumai, ka Palembang, terus deui ka Jawa nu ahirna bisa nepi ka Banyuwangi. Mugi-mugi nagara Indonesia teh boga danana, ceunah ceuk salahsahiji pajabat kareta api nu milu dina eta pertemuan para pamingpin kareta api sa ASEAN.

Tah, kukituna, geus kabayang, umpama Indonesia atawa kota-kota nu aya di Indonesia, kaasup Jakarta, geus miboga hubungan kareta api ka nagara Tiongkok, tangtu hubungan kareta api Jakarta-London teh lain ngan sakadar ngimpi atawa ngalamun wungul, tapi bisa jadi kanyataan, sabab hubungan kareta api nagara Tiongkok ka kota London ayeuna geus aya.

19. conto babasan sunda​


Babasan atau dalam bahasa Indonesia sering disebut juga sebagai ungkapan. Babasan adalah pakeman basa atau idiom yang sudah tetap, baik susunan kata-katanya maupun artinya. Babasan biasanya hanya terdiri dari dua kata. Gabungan kata-kata yang ada dalam babasan artinya tidak sama dengan arti dari masing-masing kata yang ada dalam babasan tersebut.

Misalnya seperti babasan atau ungkapan yang ada dalam bahasa Indonesia, kambing hitam, artinya bukan kambing yang berwarna hitam, tapi artinya adalah orang yang disalahkan atau dituduh. Di bawah ini contoh babasan yang ada dalam bahasa Sunda :

1. Hampang birit artinya adalah orang yang mudah disuruh atau orang yang tidak pemalas. Contoh kalimatnya : "Eta budak teh hampang birit, gampang pisah ari dititah teh, matak pikaresepeun boga pagawe saperti manehna mah".

2. Buntut kasiran artinya adalah orang yang pelit. Contoh kalimatnya : "Maneh mah boga lanceuk, keur goreng adat teh, buntut kasiran deuih, eta atuda hese pisan lamun dipentaan sumbangan teh".

3. Pait daging artinya adalah orang yang selalu sehat atau tidak pernah terkena penyakit. Contoh kalimatnya : "Pa guru mah pait daging, asa tara kakuping teu damang nya ?".

4. Beurat birit artinya adalah orang yang susah disuruh atau pemalas. Contoh kalimatnya : "Eta budak, keur kebluk teh, beurat birit deuih, euweuh pisan araheunnana budak teh".

39. Conto biantara sunda


Assalamualaikum Wr. Wb Wilujeng Enjing Sadayana. Ibu Guru nu dipihormat sareng sadaya rerencangan, siswa kelas IV nu dipikacinta. Langkung ti payun mangga urang sanggakeun puji sareng sukur ka Allah SWT nu parantos maparin waktos ka urang sadayana dugi ka tiasa patepang raray dina ieu acara. Alhamdulillah urang tiasa kempel di ieu tempat dina kaayaan sehat wal afiat. Para wargi anu dipihormat. Dina ieu waktos, sim kuring bade ngadugikeun kumaha pentingna hirup sehat kanggo urang sadayana. Luyu jeung hadis Rasulullah, kabersihan teh sabagian tina iman. Salaku jalma nu iman, urang tangtuna moal ngalalaworakeun perkara kabersihan. Urang sadaya kedah ngajaga lingkungan sareng awak sorangan sangkan dijauhkeun tina sagala panyakit. Ku margi kitu, urang kedah ngalaksanakeun hirup sadidinten ku cara anu sehat. Meresihan awak, meresihan kuku, nyikat waos anu rutin, eta conto hirup anu sehat. Conto sanesna, miceun runtah kana tempatna, nyiduh ulah di mana wae, batuk atawa beresin kudu nutupan baham. Saupami sadidinten urang ngajaga kaberesihan awak urang bakal beresih atanapi suci tina sagala kokotor. Bersih teh hartosna bersih-sucina diri urang sareng lingkungan di sabudeureun urang. Urang kedah mikacinta kana kabersihan sangkan hate urang ge suci sareng bersih. Hadirin sadaya. Cekap sakitu nu kapihatur, pamungkas ti abdi, bilih aya basa anu kirang entep seureuhna atanapi kecap anu kirang merenanah larapna, mugi kersa ngahapunten. Hatur nuhun kana perhatosanana. Wassalamualaikum wr.wb
