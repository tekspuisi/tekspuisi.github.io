14. mengapa saat mendung di malam hari terasa panas¿¿¿


Jawabannya sederhana. Ketika awan menghitam atau mendung akan terjadi proses perubahan uap air (gas) berubah menjadi air (cair). Pada proses terjadinya mendung, ini melepaskan sejumlah panas (kalor) ke udara.

Dari mana asalnya hujan? Tentu mendung. Bila kita telusuri prosesnya, mendung atau awan itu sejatinya adalah kumpulan dari uap air hasil pemanasan sinar matahari pada laut, sungai, danau, dan tempat berkumpulnya air lain. Mirip seperti kepulan asap yang keluar saat kita membuka tutup panci yang airnya sudah mendidih.

Singkatnya, udara panas membawa lebih banyak uap air ketimbang udara yang dingin. Nah, saat udara panas atau mendung itu tadi semakin naik ke atas, akhirnya bertemulah dia dengan udara dingin.

Saat bersatu dengan udara dingin itu, mendung akan melepaskan panasnya. Dan panas itu yang kita rasakan sebelum hujan. Saat semua panas itu terlepas, pasti kita merasa udara mulai dingin, dan saat itu lah hujan akan turun. Ingat, hujan pada dasarnya adalah uap air yang mengembun

17. 1. P1 : Jika hari ini langit mendung maka hari ini akan hujan P2 : Hari ini langit mendung Kesimpulan : ......... 2. P1 : Jika hari ini langit mendung maka hari ini akan hujan P2 : Hari ini tidak akan hujan Kesimpulan : ............ 3. P1 : Jika hari ini langit mendung maka hari ini akan hujan P2 : Jika hari ini akan hujan maka doni membawa paying Kesimpulan : ..............................


1. P₁: Jika hari ini langit mendung maka hari ini akan hujan.

   P₂: Hari ini langit mendung.

   Kesimpulan: Hari ini akan hujan.

2. P₁: Jika hari ini langit mendung maka hari ini akan hujan.

   P₂: Hari ini tidak akan hujan.

   Kesimpulan: Hari ini langit tidak mendung.

3. P₁: Jika hari ini langit mendung maka hari ini akan hujan.

   P₂: Jika hari ini akan hujan maka Doni membawa payung.

   Kesimpulan: Jika hari ini langit mendung, maka Doni membawa payung.

Penjelasan dengan langkah-langkah

Penarikan kesimpulan dalam logika matematika ada 3 macam yaitu

1. Modus Ponens    

p ⇒ q

p

----------    

∴    q

2. Modus Tollens  

 p ⇒ q                    

      ~q                  

---------        

∴   ~p                  

3. Silogisme  

  p ⇒ q  

  q ⇒ r

----------

∴ p ⇒ r

Nomor 1

Diketahui

P₁: Jika hari ini langit mendung maka hari ini akan hujan. P₂: Hari ini langit mendung.

Ditanyakan

Tentukan kesimpulan dari kedua premis tersebut!

Jawab

Langkah 1

Misal

p = hari ini langit mendungq = hari ini akan hujan

Berarti kedua premis dapat kita tulis.

P₁: p ⇒ qP₂: p                            

Langkah 2

Kedua premis tersebut dapat kita simpulkan menggunakan modus ponens yaitu:

p ⇒ q

p

----------    

∴    q

Jadi kesimpulannya adalah q = "Hari ini akan hujan".

Nomor 2

Diketahui

P₁: Jika hari ini langit mendung maka hari ini akan hujan. P₂: Hari ini tidak akan hujan.

Ditanyakan

Tentukan kesimpulan dari kedua premis tersebut!

Jawab

Langkah 1

Misal

p = hari ini langit mendungq = hari ini akan hujan => ~q = hari ini tidak akan hujan

Berarti kedua premis dapat kita tulis.

P₁: p ⇒ q    P₂: ~q      

Langkah 2

Kedua premis tersebut dapat kita simpulkan menggunakan modus tolens yaitu:

 p ⇒ q                    

      ~q                  

---------        

∴   ~p

Jadi kesimpulannya adalah ~p = "Hari ini langit tidak mendung".

Nomor 3

Diketahui

P₁: Jika hari ini langit mendung maka hari ini akan hujan. P₂: Jika hari ini akan hujan maka Doni membawa payung.

Ditanyakan

Tentukan kesimpulan dari kedua premis tersebut!

Jawab

Langkah 1

Misal

p = hari ini langit mendungq = hari ini akan hujanr = Doni membawa payung

Berarti kedua premis dapat kita tulis.

P₁: p ⇒ qP₂: q ⇒ r      

Langkah 2

Kedua premis tersebut dapat kita simpulkan menggunakan modus silogisme yaitu:

p ⇒ q  

q ⇒ r

----------

∴ p ⇒ r

Jadi kesimpulannya adalah p ⇒ r yaitu "Jika hari ini langit mendung, maka Doni membawa payung".
