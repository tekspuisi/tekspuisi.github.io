menggunakan rumus

S + I = L --> S (semesta) ; I (irisan) ; L ( lainnya yang bukan semesta dan irisan )

S + I = L 
S + I = P (puisi) + C (cerpen)
40 + 12 = 23 + x
52 = 23 + x
x = 52 - 23
x = 29 orang (C)

40=(12-x)
