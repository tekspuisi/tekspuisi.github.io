23. Momentum suatu elektron awalnya sama dengan mc. Untuk memperkecil panjang gelombang de Broglie elektron ini agar menjadi setengah dari semula maka energi total elektron tersebut harus diubah menjadi .... mc^2. tolong jawab pakai cara yang lengkap yaa


*Energi kinetik (EK)*
EK = ½ m.v²

dalam soal, untuk elektron
EK = ½ m.c²

c elektron = kecepatan elektron (m/s)


*Panjang gelombang de Broglie*
λ = h / p

h = kontanta planck
h = 6,6 x 10^-34 Js
p = momentum (kg m/s)

Dalam soal diketahui momentum elektron = mc

maka,
λ = h / (mc)
c = h / (mλ)

Kita subsitusi ke EK
EK = ½ m c²
EK = ½ m (h / mλ)²
EK = ½ m h² / (m².λ²)
EK = ½ h² / (m.λ²)

terlihat EK berbanding terbalik dengan λ²
EK2 / EK1 = (λ1 / λ2)²


Diketahui :
λ1 = λ
EK1 = ½mc²

panjang gelombang de Broglie diperkecil menjadi ½ dari semula:
λ2 = ½λ

Ditanyakan : EK2

Jawab :
EK2 / EK1 = (λ1 / λ2)²
EK2 / EK1 = (λ / ½λ)²
EK2 / EK1 = 4
EK2 = 4 EK1
EK2 = 4 (½mc²)
EK2 = 2mc²


Kelas : 12
Mapel : Fisika
Kategori : Relativitas
Kata Kunci : panjang gelombang de Broglie, Energi kinetik, Energi Total

p = mₒ c
(mₒ = massa diam)
λ' = ½ λ → E' = __?

Panjang gelombang de-broglie
λ = h / p
λ berbanding terbalik dengan momentum

λ' / λ = (h/p') / (h/p)
λ' / λ = p / p'
p' = (λ / λ') p
p' = (λ / ½λ) p
p' = 2 p
p' = 2 mₒ c

Energi total
E'² = (p' c)² + (mₒ c²)²
E'² = (2 mₒ c • c)² + (mₒ c²)²
E'² = 4 (mₒ c²)² + (mₒ c²)²
E'² = 5 (mₒ c²)²
E' = √5 mₒ c². ✔️
